## Prerequisites

Install required modules with pip:

> tkinter
>> Note: tkinter is included in The Python Standard Library

```sh
pip install tk
```

> pillow

```sh
pip install pillow
```

> numpy

```sh
pip install numpy
```

> cv2

```sh
pip install opencv-python
```
