## Prerequisites

Install required modules with pip:

> [tkinter]

```sh
pip install tk
```
>> Note: tkinter is included in The Python Standard Library

> [pillow]

```sh
pip install pillow
```

> [numpy]

```sh
pip install numpy
```

> [cv2]

```sh
pip install opencv-python
```
